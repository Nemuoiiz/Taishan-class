position:relative 相對定位

使用位移屬性（top、right、bottom、left）會以自己原來的位置位移，且保留自身的空間進入定位層，位置也不會被別人佔據。

父元素沒有任何的位移屬性時，會留在原地不動，這種 relative 的作法常作為 absolute 子元素位移目標的依據。

進入定位層可以使用 z-index（元素可以遮住元素）

position:absolute 絕對定位

沒有任何位移屬性（top、right、bottom、left）時， 該元素會脫離正常流向，提升為定位層， 後方元素會依據 CSS 流動的特性、流動至絕對定位元素的下方（被絕對定位元素遮住）。

若祖先元素找不到 absolute、relative、fixed， 最後會以初始容器根源素（html）作為位移目標。

父元素會用 relative 與子元素 absolute 搭配的原因是， 父元素可以維持再原來的位置，而 absolute、fixed 會脫離正常流向。
