📦 FlexBox

定義：一種排版系統，須設定兩個部分，彈性容器（flex container）、彈性項目（flex items），藉由決定主軸、次軸去對彈性項目做對齊。

📦 彈性盒設定

1. display:flex

設定父元素 display:flex 為彈性容器，子元素變成彈性項目。
彈性容器、彈性項目有各自屬性可以設定。

2. flex-direction 決定軸向

彈性盒布局是以主軸、次軸為基礎，flex-direction 決定「主軸」是水平或垂直。
決定主軸方向後，次軸則為另一方向。
row 主軸是水平排列，column 主軸是垂直排列。

2. flex-grow 彈性伸展係數

預設 0 時為不伸展
設成 1 或其他數字時，會將剩餘空間進行比例分配給彈性項目。
例如三欄 flex-grow 分別設定 class 1、2、3，空間分配則為 1/6、2/6、3/6
flex-grow: 0 等於 flex: 0 1 auto，也等於 flex: inital
flex-grow: 1 等於 flex: 1 1 auto，也等於 flex: auto

3. flex-shrink 彈性收縮係數

彈性項目收縮係數預設 1，防止所有彈性項目加起來總寬度、超過彈性容器寬度而產生的爆版現象。
flex-shrink: 1 不會爆版（預設值）。
flex-shrink: 0 當所有彈性項目加起來總寬度、超過彈性容器寬度，會產生爆版。

🔻 flex:1 與 flex:auto 差異

🔶 flex:1 => flex: 1 1 0%

真正的等寬（等高），flex-basis: 0%，無法設定寬（高）。
內容超過寬度時無法撐開寬高，會直接換行。
